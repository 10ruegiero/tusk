## Install

Download the latest version from the [releases](https://github.com/champloohq/tusk/releases/latest) page or follow the links below.

### MacOS

[**Download**](https://github.com/champloohq/tusk/releases/latest) the `.dmg` file.

### Linux

[**Download**](https://github.com/champloohq/tusk/releases/latest) the `.AppImage` or `.deb` file.

### Windows

[**Download**](https://github.com/champloohq/tusk/releases/latest) the `.exe` file.

## Features

### Keyboard shortcuts

Description            | Keys
-----------------------| -----------------------
Toggle window menu     | <kbd>Alt</kbd>
Return to Notes        | <kbd>Esc</kbd>
New note               | <kbd>Cmd/Ctrl</kbd> <kbd>N</kbd>
Delete note            | <kbd>Cmd/Ctrl</kbd> <kbd>D</kbd>
Add shortcut           | <kbd>Cmd/Ctrl</kbd> <kbd>S</kbd>
Set reminder           | <kbd>Cmd/Ctrl</kbd> <kbd>E</kbd>
Search notes           | <kbd>Cmd/Ctrl</kbd> <kbd>F</kbd>
Toggle focus mode      | <kbd>Cmd/Ctrl</kbd> <kbd>M</kbd>
Exit focus mode        | <kbd>Cmd/Ctrl</kbd> <kbd>K</kbd>
Toggle Settings        | <kbd>Cmd/Ctrl</kbd> <kbd>,</kbd>
Bold text              | <kbd>Cmd/Ctrl</kbd> <kbd>B</kbd>
Italic text            | <kbd>Cmd/Ctrl</kbd> <kbd>I</kbd>
Underline text         | <kbd>Cmd/Ctrl</kbd> <kbd>U</kbd>
Toggle checkbox        | <kbd>Cmd/Ctrl</kbd> <kbd>T</kbd>
Code block             | <kbd>Cmd/Ctrl</kbd> <kbd>W</kbd>
Toggle shortcuts       | <kbd>Cmd/Ctrl</kbd> <kbd>Shift</kbd> <kbd>S</kbd>
